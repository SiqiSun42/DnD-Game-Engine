# DnD Game Engine - UI设计说明

## 整体布局

- 界面参考设计图文件夹

- 左侧侧边栏（固定宽度，可折叠，内容：创建游戏+存档列表+咨询城主+切换账号、外观）
- 中间对话窗口（主要游戏区域）

## 侧边栏设计

 - 所有选框鼠标悬停时都会颜色变深，点击时会变得更深

### 顶部

- **"Dungeon Master"** 标题，点击可返回主菜单占位符界面

### 创建游戏

- **➕ 新的冒险** - 点击跳转到"创建新游戏"占位符界面
- **📋 显示有模板** - 点击跳转到"选择模板"占位符界面

### 存档列表

- 🔍 搜索框
- 显示所有已保存游戏的名称，按照最后创建/顺序排列
- 每个存档可点击，点击后跳转到游戏界面（跳转到“占位符界面-存档A/B/...N”）
- 鼠标悬停在存档上时，会出现右侧三个竖着的点。点击这个小图标会出现选框，有四个选项，固定（固定后存档前面会出现大头针图标，颜色变深，和鼠标悬停类似，并固定在列表顶端），复制（复制后会在下面出现一个自动命名为同名-副本的存档），删除（点击删除后会弹出一个占据全屏的窗口，询问是否确认删除），重命名。这个设计整体和窗口式ai类似.
- 存档列表的高度固定，在创建游戏之下，咨询城主之上。如果存档数量太多，会在右侧出现滚动条。

### 底部选框

- **❓ 咨询城主** - 点击打开规则查询占位符窗口
- **👤 开发者/游客** - 开发者模式选项，点击会出现选框，切换开发者/游客模式。改变头像（开发者为A，游客为G），默认是开发者。
- **☀️/🌛 切换外观** - 在开发者头像的右侧同行，点击出现选框，可以选择黑夜/白天模式的配色。默认为黑夜模式（低饱和度的深蓝色，所有字体为白色或者浅灰色）。

## 上方操作栏

- **标签页列表**（从左到右）
  - 🎒背包 - 展开右侧背包面板
  - ⚔状态 - 展开右侧角色状态面板
  - 👤人物 - 展开右侧人物笔记面板
  - 🌏世界 - 展开右侧地点笔记面板
  - 📒笔记 - 展开右侧情节笔记面板
  - ⚙设置 - 展开右侧设置面板

- **交互**
  - 点击任一标签页，下册展开栏显示对应内容占位符；再次点击收起
  - 展开栏直接遮挡对话窗口界面，对话窗口界面不动
  - 同时只能显示一个标签页内容
  - 选中效果、颜色变化同侧边栏

## 对话窗口详细设计

### 整体结构

对话窗口分为两个区域，与普通窗口式对话AI相似：

1. **对话区**
   - 显示AI扮演的DM/NPC与玩家的对话
   - 对话气泡或卡片式显示
   - 支持滚动查看历史

3. **输入框区**
   - 输入框，支持回车发送

### 右侧补充元素

- **滚动条** - 当对话超过一页时，自动出现滚动条。平时隐藏，滚动时出现。和普通网页类似。

### 交互细节

- 对话窗口自动滚动到最新消息，一般来说是DM，因为玩家输入后DM会自动回复
- 玩家可手动滚动查看历史
- 输入框获得焦点时高亮
- 发送后输入框清空，焦点保留
- 支持Shift+Enter换行
