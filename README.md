# DnD Game Engine

一个支持长期记忆和灵活故事推进的AI龙与地下城游戏系统。

## 项目描述

DnD Game Engine 是一个AI驱动的龙与地下城游戏引擎。玩家扮演一个或多个角色在动态生成的世界中冒险，AI扮演NPC和地下城主（DM）。与传统的AI聊天不同，这个系统强调：

- **长期记忆管理**: 系统记住所有对话和关键信息，DM能够调用这些信息保持情节一致，避免逻辑矛盾
- **灵活的故事推进**: 玩家可以通过多种方式推进剧情，系统能够通过节点保证故事进行，但在此基础上保留了最大的自由度
- **规则驱动的战斗**: 使用RAG调用规则书验证玩家动作的合理性，通过骰子工具保证公平的随机性

## 最终目标

完成一个功能完整的DnD游戏平台，具有以下能力：

1. **自定义和模板系统** - 玩家可以自定义游戏世界、人物、NPC，或直接使用预制故事模板开始游戏
2. **文本交互界面** - 类似ChatGPT/Claude的对话式游戏体验，拥有专门的游戏信息面板（背包、任务、笔记等）
3. **战斗系统** - 整合规则书查询和骰子，确保战斗既公平又合理
4. **长期记忆系统** - 自动保存对话、生成笔记，通过RAG智能补全矛盾信息（如NPC的性格和目标）
5. **动态任务推进** - 有软性和硬性任务，玩家可以自由探索但也能投入有伏笔和暗线的复杂情节
6. **NPC行为一致性** - NPC能够记忆过去的交互，调用自己的目标和背景故事，做出合理的反应

## 核心功能

- 支持自定义游戏设置（背景、人物、NPC、游戏长度等）和预制模板
- 文本为主的游戏玩法，配合UI面板显示游戏状态
- 规则驱动的战斗系统（RAG规则查询 + 骰子随机性）
- "咨询城主"窗口用于规则查询
- 完整的对话历史和智能笔记系统
- 开发者模式（显示所有隐藏线索和内部逻辑）

## 技术栈

| 功能 | 技术 |
|------|------|
| AI对话 | DeepSeek API |
| 规则查询 | RAG (D&D规则书向量检索) |
| 随机性 | 骰子API/伪随机数生成 |
| 长期记忆 | 本地存储 + RAG向量检索 |
| 前端 | Web (React/Vue 等) |
| 后端 | Python/Node.js (API调用和逻辑处理) |
