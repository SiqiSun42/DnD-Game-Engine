# 新手村故事 - DM指导文档

## 游戏设置

### 故事背景
主角和队友来到小渔村附近，准备补充物资。村庄附近最近被哥布林骚扰，威胁着村民的安全。

### 游戏目标
击败哥布林大王，解决村庄的问题。

---

## 人物设定

### 主角（玩家控制）
- **名字**: 凯文
- **职业**: 战士
- **简单背景**: 流浪冒险者

### 队友 NPC - 艾琳（善良守序牧师）
- **名字**: 艾琳
- **职业**: 牧师
- **种族**: 精灵
- **性格**: 善良、守序、关心弱者、经常给出建议和劝阻
- **对话风格**: 温和、体贴、经常说"我们应该..."
- **关键行为**:
  - 第一场战斗中，如果玩家躲避，会主动出战帮助村民
  - 在码头试图离开时，会劝阻："我们还没帮助这些人啊..."
  - 做不道德的事时，会皱眉表达不满
  - 在洞穴探索时，会提示检查环境

---

## 游戏流程

### 初始场景：村庄入口

**场景描述**: 
玩家和艾琳来到小渔村附近，准备补充物资。突然看到几个哥布林正在抢劫村民。

**玩家选择**:
1. 帮助村民 → 触发战斗
2. 躲起来 → 艾琳主动出战
3. 离开 → [暂时不考虑]

**战斗系统** (暂时留空):
- 玩家输入战斗结果: `胜利` / `失败` / `逃跑`
- `胜利`: 村民感谢，推荐去找村长，可能给面包
- `失败`: 玩家损失一半金币，村民仍感谢，艾琳建议提升技能
- `逃跑`: 无后果（艾琳在场景时会主动战斗）

**战斗后**:
- 村民告知村庄最近在闹哥布林
- 推荐去村长处接任务
- **解锁: 村庄任务**

---

### 村庄任务 (可选，各可独立完成)

#### 任务1: 从村长处获得信息
- **触发地点**: 村庄 - 村长办公室
- **NPC**: 村长
- **内容**: 
  - 村长讲解哥布林情况
  - 玩家可领取"击败哥布林大王"任务
  - 村长提供地图（标记南边洞穴位置，但有陷阱标记不准）
- **状态**: 完成后显示 `[已完成]`

#### 任务2: 探索村庄
- **地点**: 村庄（多个位置）

**商店**:
- 可购买: 食物、火把、药品等
- 可出售: 任何物品
- NPC: 商人（中立，仅做生意）

**酒馆**:
- 可购买: 饮料、食物
- 可聊天: NPC闲聊（暂时无实际影响）
- NPC: 酒保（暂时无特殊剧情）

**码头**:
- 可结束游戏: 玩家选择离开
  - 艾琳会劝阻: "我们还没打败哥布林大王啊..."
  - 玩家坚持可离开 → `游戏结束: 你们踏上了新的旅途...`
- NPC: 渔民（中立）

**随机NPC**:
- 可聊天互动（暂时无实际影响）

#### 任务3: 找到哥布林洞穴
- **触发条件**: 可在村庄任何时刻开始探索
- **方向选择**:
  - **南边的路** → 哥布林洞穴（正确方向）
  - **北边的路** → 死路（无后果，可返回）
- **获得信息方式**:
  - 看村长的地图
  - 询问村民
  - 自己探索发现
- **状态**: 发现洞穴位置后显示 `[已完成]`
- **解锁**: 哥布林洞穴任务

---

### 哥布林洞穴任务 (线性推进)

#### 前置条件
- 必须有火把或魔法照明
  - 火把: 商店购买
  - 魔法: 法师职业可用魔法照明（暂时不考虑）
- 艾琳会提示: "这里太黑了，我们需要火把..."

#### 洞穴战斗序列

**战斗1: 中等难度哥布林**
- 玩家输入: `胜利` / `失败` / `逃跑`
- 胜利: 继续进入洞穴深处
- 失败/逃跑: 返回村庄或洞穴入口，下次进入重新触发此战斗

**战斗2: 高等难度哥布林**
- 前置: 必须战胜战斗1
- 玩家输入: `胜利` / `失败` / `逃跑`
- 胜利: 进入岔路
- 失败/逃跑: 返回或重新进入，再次遭遇此战斗

#### 岔路抉择

**场景描述**: 战胜高等难度哥布林后，发现左右两条通道。

**艾琳的提示**: "让我仔细看看...这条路有点不对劲。我们应该小心。"

**玩家选择**:

**选项A: 走左路** (危险)
- 首次选左: 
  - 触发陷阱，损失1/3血量
  - 哥布林大王占先手战斗
- 失败过一次后再选左: 
  - 直接触发哥布林大王战斗（陷阱已知）

**选项B: 走右路** (安全)
- 直接触发哥布林大王的正常战斗

#### 哥布林大王战斗

**战斗3: 哥布林大王** (Boss战)
- 玩家输入: `胜利` / `失败` / `逃跑`
- 胜利: 大王被击败，进入后续剧情
- 失败/逃跑: 返回村庄或重新进入洞穴，再次遭遇

#### 战斗后抉择

**大王被击败后**:
- 大王求饶: "别杀我！我告诉你个秘密！"

**玩家选择**:
1. **听秘密** → 大王透露与村长的勾结
   - 对话内容: 大王说明与村长如何串通的过程
   - 解锁: 返回村庄时可质问或处置村长
   
2. **直接杀死** → 无法获得秘密信息
   - 艾琳可能表达意见（若为善良性格）
   - 返回村庄后村长不会承认勾结

**游戏推进**: 可选择返回村庄或继续探索洞穴

---

### 返回村庄（可选）

#### 返回前的状态
- **未击败大王**: 村民/队友表示鼓励，无特殊剧情
- **失败过一次，再次回村**: 
  - 可质问村长: "为什么你的地图有陷阱？"
  - 高武力值可威胁村长说出真相
  - 艾琳建议继续挑战

#### 击败大王后返回
- **村民反应**: 欢呼感谢，所有商品打5折
- **村长反应** (取决于玩家是否知道勾结):
  - 若知道勾结: 
    - 可选告发、威胁、接受贿赂等
    - 艾琳会根据选择表达态度
  - 若不知道: 村长正常感谢

#### 游戏结束
- **正常结束**: 击败大王，返回村庄
  - 村民感谢，获得报酬
  - 可选继续探索或离开（码头）
  
- **早期离开**: 在击败大王前去码头
  - 艾琳劝阻，但玩家可坚持
  - 显示: `游戏结束: 你们踏上了新的旅途...`

## DM提示

### 关键记忆点
- 保持村长的可疑性（但直到战胜大王前不暴露）
- 艾琳在关键时刻给出建议，但尊重玩家选择
- 所有任务完成后标记 `[已完成]`，避免重复触发

### 玩家状态追踪
- 当前位置
- 已完成任务列表
- 当前金币/物品
- 血量状态
- 是否知道村长勾结秘密

### 战斗系统暂时处理
- 接受玩家的 `胜利` / `失败` / `逃跑` 输入
- 根据输入自动调用对应结局
- 后续完成实际战斗系统后替换此机制
